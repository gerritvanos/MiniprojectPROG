Om de Reisinformatie Applicatie te kunnen gebruiken is Python 3.6.2 (of recenter) nodig. Dit kunt u downloaden op de volgende pagina:
https://www.python.org/downloads/

Wanneer u dubbelklikt op het script (NS_Reistijden_Info_Applicate) word het script geopend.

Overigens, wanneer er een IDE geÃ¯nstalleerd (bijvoorbeeld Pycharm) word het script via Pycharm geopend. Dit is mogelijk maar niet gewenst. Om dit te voorkomen volgt u de volgende stappen:

1. Klik met rechtermuisknop op het script.
2. Navigeer met uw muiscursor naar Openen met >
3. Selecteer uit het menu wat verschijnt Python 3.6.2.

Na deze stappen zal het script een scherm openen en vervolgens de API van de NS.

Voor het correct weergeven van het verlag(Verslag_V1N_team3.docx) is het nodig om het lettertype 'Frutiger Bold Regular' nodig.
Dit is te downloaden via de volgende link: https://www.wfonts.com/font/frutiger
Het bestand opent uiteraard wel maar word zal dan zelf een ander lettertype selecteren.
